# backend_meli

## Built with 🛠️

- Node
- Express
- Axios

## Instalation back

1. **Clone Project**: 
```
git clone https://github.com/rafeldev/backend_meli
```

2. **Move to project**: 
```
cd backend_meli
```

3. **Install dependencies**: 
```
npm install
````

3. **execute program (this program start in port 3001)**: 
```
nodemon dist
```
